# 急救知识指南（First Aid Knowledge Guide）

> 人人都能学会的家庭与公共场所急救手册

## 在线阅读

GitHub Pages 地址：[https://zhangwei89.github.io/First-Aid-Knowledge-Guide/](https://zhangwei89.github.io/First-Aid-Knowledge-Guide/)

## 本书简介

一本面向普通公众的急救科普读物，涵盖 10 篇 43 章正文 + 7 个附录，约 7.7 万字。

- **场景覆盖**：家庭、学校、职场、户外、灾害现场
- **技能覆盖**：CPR、AED、海姆立克、止血、创伤处理、常见急症、特殊人群
- **结构设计**：每章按"适用场景—判断方法—操作步骤—注意事项—常见错误"展开
- **附录工具**：10 张流程图、80 个误区、25 个速查页、50 道自测题

## 章节结构

- 第一篇 急救基础知识（4 章）
- 第二篇 心肺复苏与生命支持（4 章）
- 第三篇 创伤急救（5 章）
- 第四篇 常见急症处理（6 章）
- 第五篇 常见疾病急救（6 章）
- 第六篇 特殊人群急救（4 章）
- 第七篇 特殊环境急救（5 章）
- 第八篇 灾害与突发事件应对（5 章）
- 第九篇 急救技能训练（4 章）
- 第十篇 附录（7 个）
