# 《急救知识指南》核心设定

> 本文档对应网文写作流程的 Phase 2（核心设定），目的是把全书的"读者/语气/深度/医学边界"一次性锁死，避免后续 43 章写到一半整体翻工。

## 基本信息

- **书名**：《急救知识指南——人人都能学会的家庭与公共场所急救手册》
- **题材**：非虚构 / 实用工具书 / 公众急救科普
- **目标平台**：纸书 / 电子书 / 公众号长文（图文结合 + 流程卡）
- **预计字数**：正文约 30-35 万字（含附录与速查手册）
- **目标读者**：普通大众（家庭成员、职场白领、教师、社区志愿者），零医学基础也能看懂

## 一句话定位

> 一本能让你在紧急时刻翻到就能用的急救手册——既有原理、也有动作、还有边界。

## 写作风格

| 维度 | 标准 |
|------|------|
| 语气 | 冷静、专业、可信任；避免说教与恐吓 |
| 句长 | 短句优先（不超过 25 字/句）；关键步骤再拆更短 |
| 术语 | 首次出现的医学术语用通俗解释+原词括号；不回避术语，但要求"读完即懂" |
| 第二人称 | 主用"你"；多步骤场景用"施救者"统一称呼 |
| 章节开头 | 不堆天气/场景/悬念；用"学完本章你能做 X"切入 |
| 章节结尾 | 一句话总结 + 一句"什么时候必须拨打 120"或"什么时候必须就医" |
| 数据 | 关键数字（如黄金 4 分钟、按压深度 5-6 cm）以"⚠️ 关键数字"块单独标注 |
| 流程 | 用编号列表 + 流程图描述符（便于后续插画） |
| 错误 | 每个实操章节末尾用"❌ 常见错误"块列 3-5 条 |

## 医学边界（红线）

1. **不替代专业医疗**：每章必须出现一次"本操作不能替代医生/急救人员"的边界提示。
2. **药品不点名推荐**：仅描述常见 OTC 类别（如"解热镇痛药""抗过敏药"），不点名具体品牌；剂量给区间不给处方。
3. **创伤严重度分流**：明确告诉读者"什么情况必须立刻 120"，避免延误。
4. **免责语**：前言和附录统一放置免责声明。
5. **来源标注**：所有关键数字/操作步骤对应 AHA《2020 心肺复苏与心血管急救指南》、中国红十字会《救护员指南》、ERC《2021 Guidelines》其中之一；附录汇总参考清单。

## 不做的事

- 不讲冷门奇难病例。
- 不写人物故事或煽情段落。
- 不引用未经验证的"网传偏方"。
- 不暗示任何急救手法能治本病。
- 不为任何药品/器材品牌背书。

## 章节统一模板

每个标准章节应包含以下结构（顺序固定）：

1. 学习目标（2-3 条）
2. 适用场景（1 段铺垫）
3. 一、核心概念（200-400 字，建立认知）
4. 二、判断方法（步骤化清单 + 怎么做/看哪里）
5. 三、操作步骤（编号步骤，每步 ≤ 50 字，关键动作加粗）
6. 四、注意事项（含 ⚠️ 关键数字 块、⚠️ 必须避免 块）
7. 五、常见错误（❌ 常见错误 块，3-5 条）
8. 本章小结（一句话）
9. 何时必须拨打 120（2-3 条标准）

字数目标：

- 基础章节（概念+判断）：约 3000-3500 字
- 实操章节（CPR/AED/止血/海姆立克等）：约 4000-5000 字
- 短章节（附录/小知识）：约 1000-1500 字

## 流程卡规范

每个核心操作（≥7 个步骤或 ≥2 种判断分支）必须配一张流程卡，文本格式如下：

```
【流程卡】CPR 操作流程
1. 确认环境安全 → 2. 判断意识（拍肩呼喊） → 3. 呼救（指定一人拨打 120+取 AED）
   → 4. 检查呼吸（≤10 秒） → 5. 胸外按压 30 次 → 6. 开放气道 → 7. 人工呼吸 2 次
   → 8. 持续按压：通气比 30:2 → 9. AED 到场立即使用 → 10. 持续至专业人员接手
```

## 文件组织

```
急救知识指南/
├── 设置/                  # 本文档 + 写作模板 + 医学参考
├── 大纲/                  # 全书大纲 + 各篇细纲
├── 正文/                  # 各篇正文 + 前言 + 附录
└── 追踪/                  # 进度 + 写作日志 + 归档
```

## 协作约定

- 每完成 5 章做一次自检（章节字数 / 模板一致性 / 红线扫描）。
- 完成一篇后做一次中期检查（章节衔接 / 术语统一 / 流程卡完整性）。
- 全书完成后做总检（数字一致性 / 参考标注 / 免责语 / 自测题答案）。
