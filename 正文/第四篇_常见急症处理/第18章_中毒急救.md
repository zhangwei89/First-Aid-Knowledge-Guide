# 第 18 章 中毒急救

## 学习目标

学完本章之后，你能够：

1. 区分食物中毒、药物中毒、农药中毒、CO 中毒、化学品中毒。
2. 立即识别中毒表现。
3. 用正确方法催吐（部分情况）。
4. 知道 CO 中毒要立即通风。
5. 保留毒物样本供医生参考。

## 适用场景

中毒事件类型多、变化快。本章按最常见的几类中毒展开，建立"识别 + 立即处理 + 立即就医"的基本思路。

## 一、中毒的常见类型

| 类型 | 常见场景 | 危险程度 |
|------|----------|----------|
| 食物中毒 | 过期食品、野生菌、未煮熟食物 | 中-高 |
| 药物中毒 | 过量服药、儿童误服 | 中-高 |
| 农药中毒 | 误服、自服、皮肤接触 | 高 |
| CO 中毒 | 煤炉、燃气热水器、密闭车内 | 高 |
| 化学品中毒 | 清洁剂、油漆、工业化学品 | 高 |

## 二、识别要点

大多数中毒有相似的早期表现：

- 恶心、呕吐、腹痛、腹泻。
- 头晕、乏力、意识模糊。
- 呼吸困难、口唇发紫。
- 瞳孔异常（散大或缩小）。
- 出汗、流涎、皮肤潮红或苍白。

特殊表现提示特定类型：

- **CO 中毒**：口唇樱桃红色（典型）。
- **有机磷农药**：流涎、出汗、瞳孔缩小、肌束震颤。
- **阿片类药物**：呼吸抑制、瞳孔针尖样缩小。
- **酒精中毒**：嗜睡、呕吐、瞳孔散大。
- **野生菌中毒**：幻觉、肝肾损伤（部分菌种）。

## 三、现场处理原则

> 【流程卡】中毒急救五步
> 1. 立即脱离中毒环境（CO、化学品）
> 2. 评估意识与呼吸
> 3. 必要时 CPR
> 4. 保留毒物样本（呕吐物、药瓶、食物）
> 5. 立即 120 + 中毒控制中心（010-83132345）

**核心原则**：

- **先脱环境、再处理**：CO、化学品必须先把病人移出中毒环境。
- **意识清醒、能配合 → 催吐**：用手指、筷子、勺柄刺激咽部。
- **意识障碍 → 不要催吐**：可能误吸。
- **保留样本**：呕吐物、剩余食物、药瓶、说明书、容器全部保留交给医生。
- **立即拨打 120**：同时拨打中毒控制中心（010-83132345）咨询。

## 四、各类中毒具体处理

**食物中毒**

1. 意识清醒、刚进食 1-2 小时内 → 催吐。
2. 让病人大量饮水（淡盐水），再催吐，反复几次。
3. 不要给止泻药（让毒素排出）。
4. 严重呕吐、腹泻 → 口服补液盐防脱水。
5. 集体发病或怀疑野生菌、河豚、农药污染 → 立即 120。

**药物中毒（安眠药、镇痛药、降压药等）**

1. 意识清醒、刚服药 → 催吐。
2. 保留药瓶和说明书交给医生。
3. 意识障碍 → 复苏体位，立即 120。
4. 不要自行"灌水稀释"或"灌醋"。

**农药中毒（有机磷、拟除虫菊酯等）**

1. 立即脱去污染衣物。
2. 用大量清水冲洗皮肤（至少 15 分钟）。
3. 误服 → 意识清醒时催吐。
4. **绝对禁用**：用肥皂水或油类导泻（加重吸收）。
5. 立即 120 + 中毒控制中心。

**CO 中毒（一氧化碳）**

1. **第一动作：开窗通风、切断 CO 来源**。
2. 把病人立即移到空气新鲜处。
3. 拨打 120。
4. 意识障碍但有呼吸 → 复苏体位。
5. 无呼吸 → 立即 CPR。
6. **所有 CO 中毒者必须送医**——可能需要高压氧治疗。

**化学品中毒**

1. 误服：
   - 不要催吐（强酸强碱催吐会二次损伤食道）。
   - 不要中和（化学反应产热加重伤害）。
   - 立即 120 + 中毒控制中心。
2. 皮肤接触：
   - 用大量清水冲洗至少 15 分钟。
   - 脱去污染衣物。
3. 吸入：
   - 立即移到空气新鲜处。
   - 必要时 CPR。

**酒精中毒**

1. 轻度（嗜睡、能唤醒、走路不稳）：
   - 让病人侧卧（防止呕吐误吸）。
   - 保暖。
   - 观察。
2. 重度（昏迷、呼吸抑制、瞳孔散大）：
   - 复苏体位。
   - 必要时 CPR。
   - 立即 120。

**野生菌中毒**

1. 立即催吐（清醒时）。
2. 保留剩余菌种供医生辨认。
3. 立即 120。
4. 部分菌种有"假愈期"，中毒后症状缓解但数天后再次恶化，必须住院观察。

## 五、不要做的事

> **❌ 中毒常见错误**
> - 意识不清时催吐 → 误吸窒息。
> - 用土方灌牛奶、灌醋、灌绿豆汤 → 无效、延误就医。
> - 给病人灌大量水 → 加重呕吐、电解质紊乱。
> - 用手指抠喉咙挖异物 → 可能损伤咽喉。
> - 强酸强碱催吐 → 二次损伤。
> - 单独留下意识不清者 → 可能窒息。

## 六、何时必须 120

满足下列任意一条，立即拨打 120：

- 任何意识障碍。
- 任何呼吸抑制。
- 任何抽搐。
- 怀疑农药、CO、药物过量。
- 集体中毒。
- 儿童、孕妇、老年人中毒。
- 任何让你"心里没底"的中毒。

## 七、预防

- 药品、化学品放在儿童拿不到的地方（高处 + 锁柜）。
- 过期药品定期清理。
- 不采食野生菌、不吃河豚。
- 不吃变质、过期食品。
- CO 报警器安装在厨房、卧室（特别是用煤炉、燃气热水器的家庭）。
- 燃气热水器安装位置合规（不安装在浴室内）。
- 室内密闭环境不用炭火取暖。

## 本章小结

中毒急救核心：脱环境、保留样本、立即就医。不要盲目催吐（强酸强碱、意识障碍不要催吐）。CO 中毒先通风。中毒控制中心电话 010-83132345。

## 何时必须拨打 120

满足下列任意一条，立即拨打 120：

- 意识障碍。
