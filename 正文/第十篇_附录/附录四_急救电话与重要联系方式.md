## 九、紧急情况下的多方联系

### 多人伤员事件

- 110（治安 / 现场指挥）
- 119（火灾 / 救援）
- 120（医疗急救）
- 单位 / 学校安全管理部门
- 伤员家属

### 重大事故 / 灾难

- 110 / 119 / 120（基本急救）
- 应急管理部门
- 当地政府
- 保险公司
- 媒体应对部门（单位）

### 涉外事件

- 当地外事部门
- 旅行社 / 单位外事部门
- 大使馆 / 领事馆

## 十、应急通讯方式

### 常规通讯

- 手机通话。
- 短信 / 微信。
- 紧急联系人手机。

### 应急通讯

- 卫星电话（户外、灾区）。
- 对讲机（短距离、户外）。
- 应急广播（接收信息）。

### 特殊情况

- 听障人士：12110 报警短信、视频报警。
- 老人：一键呼叫手机、定位手表。
- 儿童：儿童手表、定位器。
- 旅外：手机漫游、当地 SIM 卡。

## 十二、应急号码的备用方案

### 主线路不通

- 备用号码（如 110 不通时用 112 国际通用）。
- 邻居、单位协助。
- 微信 / 短信联系。

### 手机没电

- 备用电池 / 充电宝。
- 借他人手机。
- 求助公用电话（越来越少）。

### 网络中断

- 短信 / 2G 网络仍可发短信。
- 应急广播。

## 本附录核心要点

1. **核心号码**：120（医疗）、119（火警/救援）、110（治安）、122（交通）。
2. **专科咨询**：12320（卫生健康）、12345（政府服务）。
3. **个人急救卡**：每个家庭成员 + 高风险人群随身携带。
4. **家庭联系卡**：贴在急救箱盖内侧、家庭门口。
5. **应急通讯**：手机、短信、应急广播多重保障。
