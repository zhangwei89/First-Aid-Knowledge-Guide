# 附录一 常用急救流程图

> 本附录汇集本书出现的核心急救流程图，可单独打印张贴。

## 一、心肺复苏（CPR）流程

> **适用：成人 / 儿童 / 婴儿（意识丧失 + 无呼吸）**

```
                ┌──────────────┐
                │  发现倒地者  │
                └──────┬───────┘
                       ↓
                ┌──────────────┐
                │ 确认环境安全 │
                └──────┬───────┘
                       ↓
                ┌──────────────┐
                │ 判断意识     │
                │ 拍肩呼喊     │
                └──────┬───────┘
                       ↓
                  ┌────┴────┐
                  ↓         ↓
            ┌─────────┐  ┌──────────┐
            │ 无反应  │  │  有反应   │
            └────┬────┘  └──────────┘
                 ↓
         ┌───────────────┐
         │  呼救：指定人 │
         │ 拨打 120 + 取 AED │
         └───────┬───────┘
                 ↓
         ┌───────────────┐
         │  检查呼吸     │
         │  ≤ 10 秒      │
         └───────┬───────┘
                 ↓
            ┌────┴────┐
            ↓         ↓
      ┌──────────┐ ┌──────────┐
      │ 有呼吸  │ │ 无呼吸   │
      │ 复苏体位│ │ 立即 CPR │
      └──────────┘ └──────┬───┘
                           ↓
                  ┌────────────────┐
                  │ 胸外按压 30 次 │
                  │  成人 5-6 cm   │
                  │  100-120 次/分 │
                  └────────┬───────┘
                           ↓
                  ┌────────────────┐
                  │ 开放气道       │
                  │ 仰头举颏法     │
                  └────────┬───────┘
                           ↓
                  ┌────────────────┐
                  │ 人工呼吸 2 次  │
                  │ 每次 1 秒      │
                  └────────┬───────┘
                           ↓
                  ┌────────────────┐
                  │ 30:2 持续      │
                  │ 5 个循环 ≈ 2 分钟 │
                  └────────┬───────┘
                           ↓
                  ┌────────────────┐
                  │ AED 到场立即用  │
                  └────────┬───────┘
                           ↓
                  ┌────────────────┐
                  │ 持续至专业人员 │
                  │ 接手或病人恢复 │
                  └────────────────┘
```

## 二、AED 使用流程

> **适用：心脏骤停、AED 到场后立即使用**

```
         ┌──────────────┐
         │ 取出 AED  │
         │ 按下电源键  │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 撕开电极片  │
         │ 按图示粘贴  │
         │（右上 + 左肋下）│
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ AED 自动分析 │
         │ 不要触碰病人 │
         └──────┬───────┘
                ↓
           ┌────┴────┐
           ↓         ↓
    ┌─────────────┐ ┌──────────────┐
    │ 建议电击  │ │ 不建议电击 │
    └──────┬──────┘ └──────┬───────┘
           ↓                ↓
    ┌─────────────┐ ┌──────────────┐
    │确认无人接触 │ │ 立即恢复 CPR │
    └──────┬──────┘ └──────┬───────┘
           ↓                ↓
    ┌─────────────┐ ┌──────────────┐
    │ 按下电击键 │ │ 2 分钟后     │
    └──────┬──────┘ │ AED 重新分析 │
           ↓        └──────────────┘
    ┌─────────────┐
    │ 立即恢复 CPR │
    │ 2 分钟后再次 │
    │ AED 分析    │
    └─────────────┘
```

## 三、海姆立克急救流程

> **适用：完全气道梗阻（不能说话、不能咳嗽、双手掐脖）**

### 成人 / 儿童（1 岁以上）

```
         ┌──────────────┐
         │ 完全梗阻    │
         │ 双手掐脖    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 站到背后    │
         │ 双臂环抱腰部│
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 一手握拳    │
         │ 拳眼对准    │
         │ 肚脐上方    │
         │ 两横指处    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 另一手包住  │
         │ 快速向内向上│
         │ 冲击腹部    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 重复直到    │
         │ 异物排出    │
         │ 或意识丧失  │
         └──────────────┘
```

### 婴儿（1 岁以下）

```
         ┌──────────────┐
         │ 完全梗阻    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 俯卧在前臂  │
         │ 头低位      │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 拍背 5 次    │
         │ 掌根在两肩胛│
         │ 骨之间      │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 翻转仰卧    │
         │ 在另一前臂  │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 胸部冲击 5 次│
         │ 两根手指    │
         │ 胸骨下半部  │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 重复直到    │
         │ 异物排出    │
         │ 或意识丧失  │
         └──────────────┘
```

## 四、止血流程

> **适用：外伤出血**

```
         ┌──────────────┐
         │ 出血部位    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 戴一次性手套│
         │ 保护自己    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 清洁纱布    │
         │ 覆盖伤口    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 直接加压    │
         │ 持续 5-10 分钟│
         │ 不松开      │
         └──────┬───────┘
                ↓
           ┌────┴────┐
           ↓         ↓
     ┌──────────┐ ┌──────────────┐
     │ 出血停止 │ │ 出血不止    │
     │ 加压包扎 │ │              │
     └──────────┘ └──────┬───────┘
                          ↓
                  ┌──────────────┐
                  │ 肢体大出血  │
                  │ 无法压迫    │
                  └──────┬───────┘
                         ↓
                  ┌──────────────┐
                  │ 止血带      │
                  │ 仅限极端情况│
                  │ 记录时间    │
                  └──────┬───────┘
                         ↓
                  ┌──────────────┐
                  │ 立即 120     │
                  └──────────────┘
```

## 五、烧伤处理流程

> **适用：火焰、热液、蒸汽烫伤**

```
         ┌──────────────┐
         │ 立即脱离    │
         │ 热源        │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 流动凉水    │
         │ 冲洗 15-20 分│
         │ 10-15℃     │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 在凉水中    │
         │ 去除衣物    │
         │（不撕扯）   │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 继续浸泡    │
         │ 10-30 分钟  │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 清洁纱布    │
         │ 覆盖        │
         └──────┬───────┘
                ↓
           ┌────┴────┐
           ↓         ↓
     ┌──────────┐ ┌──────────────┐
     │ Ⅰ 度     │ │ Ⅱ 度及以上 │
     │ 浅 Ⅱ 度 │ │ 或大面积   │
     └────┬─────┘ └──────┬───────┘
          ↓              ↓
    ┌──────────┐ ┌──────────────┐
    │ 在家处理 │ │ 立即 120     │
    └──────────┘ └──────────────┘

⚠️ 禁忌：
- 不要涂牙膏、酱油、油
- 不要弄破水疱
- 不要冰敷
```

## 六、中暑处理流程

> **适用：热痉挛、热衰竭、热射病**

```
         ┌──────────────┐
         │ 高温环境    │
         │ 中暑症状    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 立即转移至  │
         │ 阴凉通风处  │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 脱去多余衣物│
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 冷水擦浴    │
         │ 风扇吹      │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 冰袋大血管  │
         │ 颈/腋/腹股沟│
         └──────┬───────┘
                ↓
           ┌────┴────┐
           ↓         ↓
     ┌──────────┐ ┌──────────────┐
     │ 意识清醒 │ │ 意识障碍    │
     │ 淡盐水    │ │ 不要口服   │
     └────┬─────┘ └──────┬───────┘
          ↓              ↓
    ┌──────────┐ ┌──────────────┐
    │ 在家观察 │ │ 立即 120     │
    └──────────┘ └──────────────┘

⚠️ 热射病死亡率高：
- 体温 > 40℃
- 意识障碍
- 抽搐
- 立即降温 + 立即 120
```

## 七、120 拨打流程

```
         ┌──────────────┐
         │ 拨打 120     │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │  报告：      │
         │  地址（精确）│
         │  地标       │
         │  病人情况    │
         │  已做处置    │
         │  联系电话    │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 不要先挂电话│
         │ 等调度员确认│
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 派人到地标  │
         │ 接救护车    │
         └──────────────┘
```

## 十、骨折急救

```
         ┌──────────────┐
         │ 怀疑骨折    │
         │（外伤 + 畸形/│
         │ 剧痛/不能动）│
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 不移动      │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 止血        │
         │（开放骨折） │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 临时固定    │
         │ 夹板 + 绷带 │
         └──────┬───────┘
                ↓
         ┌──────────────┐
         │ 检查远端    │
         │ 循环/活动   │
         └──────┬───────┘
                ↓
           ┌────┴────┐
           ↓         ↓
     ┌──────────┐ ┌──────────────┐
     │ 肢体骨折 │ │ 脊柱/骨盆/髋│
     │ 立即就医 │ │ 立即 120    │
     └──────────┘ └──────────────┘

⚠️ 禁止：
- 不要复位
- 不要强行搬动
- 不要塞回骨头
```
