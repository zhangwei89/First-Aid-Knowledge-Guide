# 急救知识指南

> 人人都能学会的家庭与公共场所急救手册

## 本书简介

一本面向普通公众的急救科普读物，涵盖 10 篇 43 章正文 + 7 个附录。

- **场景覆盖**：家庭、学校、职场、户外、灾害现场
- **技能覆盖**：CPR、AED、海姆立克、止血、创伤处理、常见急症、特殊人群
- **结构设计**：每章按"适用场景—判断方法—操作步骤—注意事项—常见错误"展开
- **附录工具**：10 张流程图、80 个误区、25 个速查页、50 道自测题

## 写作约定

- 本书所有内容参考国内外权威急救指南与公众急救培训规范
- 所有关键数字均标注来源（来源 R1-R12）
- 所有药品、器材均为通用名，不涉及商业品牌
- 急救从来不只是医生的事——**愿你成为那个"在场的、敢动手的、懂方法的人"**

## 阅读方式

1. **入门读者**：从第 1 章顺序读到第 4 章，建立基础认知
2. **应急读者**：直接查阅「附录七 急救速查手册」
3. **培训读者**：配合红十字会、120 急救中心等机构组织的现场培训
