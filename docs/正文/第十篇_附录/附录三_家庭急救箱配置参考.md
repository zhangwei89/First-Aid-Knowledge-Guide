# 附录三 家庭急救箱配置参考

> 本附录提供从基础到完整的家庭急救箱配置方案，可按家庭实际需求选择。

## 一、基础版（必配）

适合：单人 / 两人家庭，无特殊慢病。

| 类别 | 项目 | 数量 |
|------|------|------|
| 药品 | 退烧止痛药 | 1 盒 |
| 药品 | 抗过敏药 | 1 盒 |
| 药品 | 口服补液盐 | 1 盒 |
| 药品 | 外用消毒剂（碘伏）| 1 瓶 |
| 药品 | 外用烫伤膏 | 1 支 |
| 器材 | 体温计 | 1 个 |
| 器材 | 创可贴 | 1 盒 |
| 器材 | 医用胶带 | 1 卷 |
| 器材 | 医用纱布 | 1 包 |
| 器材 | 医用绷带 | 1 卷 |
| 器材 | 一次性手套 | 5 副 |
| 耗材 | 酒精棉片 | 1 盒 |
| 耗材 | 碘伏棉签 | 1 盒 |
| 耗材 | 医用口罩 | 1 包 |
| 其他 | 急救联系卡 | 1 张 |

## 二、标准版（推荐）

适合：3-4 人家庭，有老人或儿童。

在基础版上增加：

| 类别 | 项目 | 数量 |
|------|------|------|
| 药品 | 止泻药 | 1 盒 |
| 药品 | 儿童退烧药 | 1 盒 |
| 药品 | 儿童止泻药 | 1 盒 |
| 药品 | 儿童退热贴 | 1 盒 |
| 药品 | 益生菌 | 1 盒 |
| 器材 | 电子血压计 | 1 个 |
| 器材 | 三角巾 | 1 条 |
| 器材 | 止血带 | 1 条 |
| 器材 | 镊子 | 1 把 |
| 器材 | 剪刀（圆头）| 1 把 |
| 器材 | 手电筒 + 电池 | 1 套 |
| 耗材 | 清洁湿巾 | 1 包 |
| 其他 | 应急手电 | 1 个 |

## 三、完整版（家庭成员有慢病 / 过敏体质）

适合：家庭成员有心脏病、糖尿病、过敏等慢性病。

在标准版上增加：

| 类别 | 项目 | 数量 |
|------|------|------|
| 药品 | 速效救心丸 | 1 瓶（医嘱） |
| 药品 | 硝酸甘油 | 1 瓶（医嘱） |
| 药品 | 阿司匹林（普通）| 1 盒（医嘱） |
| 药品 | 抗过敏药（更强）| 1 盒 |
| 药品 | 肾上腺素自动注射器 | 1 支（医嘱 + 培训） |
| 器材 | 血糖仪 + 试纸 | 1 套 |
| 器材 | 血氧仪 | 1 个 |
| 器材 | 雾化器（如有哮喘）| 1 个 |
| 器材 | 家用氧气瓶（如有条件）| 1 个 |
| 其他 | AED（如能负担）| 1 台 |

## 四、急救箱的存放

### 存放条件

| 条件 | 标准 |
|------|------|
| 温度 | 常温 10-30℃（特殊药品按说明书） |
| 湿度 | 干燥（避免浴室、厨房水槽下） |
| 光照 | 避光（避免阳台、窗边） |
| 高度 | 儿童拿不到的高处 |
| 位置 | 固定位置（全家人都知道） |
| 标识 | 明显标识（红色十字、绿色急救标志） |

### 存放位置推荐

| 位置 | 适用家庭 |
|------|----------|
| 客厅高处柜子 | 多数家庭 |
| 主卧衣柜高处 | 主卧急救 |
| 厨房高处（远离热源）| 厨房急救 |
| 书房 / 工作区 | 办公区 |

### 不推荐的存放位置

- 浴室（潮湿）。
- 厨房水槽下（潮湿）。
- 阳台（光照、温度变化）。
- 车辆后备箱（高温，特殊储存除外）。

## 六、急救箱的定期维护

### 每月检查

- 急救箱位置是否变动。
- 应急联系卡是否更新。
- 一次性物品是否齐全。

### 每季度检查（建议 1 月初、4 月初、7 月初、10 月初）

- 药品有效期（详细见下表）。
- 器材完整性。
- 电池电量（体温计、手电、血压计）。
- 一次性物品数量。

### 每年检查（建议春节前）

- 完整清单核对。
- 过期物品更换。
- 慢病药品按医生指导调整。
- AED 电极片有效期（如有）。

## 七、不同家庭场景的配置建议

### 婴幼儿家庭

**重点加强**：

- 儿童退烧药（滴剂/混悬液）。
- 儿童创可贴（尺寸小）。
- 婴儿喂药器。
- 儿童退热贴。
- 婴儿体温计（肛温 + 腋温）。

**特殊注意**：

- 1 岁以下禁用蜂蜜、感冒药、止咳药。
- 不要囤积成人药品。
- 不要使用过期儿童药品。

### 老年人家庭

**重点加强**：

- 慢病药（备用 1 周量）。
- 速效救心丸 / 硝酸甘油。
- 血压计、血糖仪。
- AED（如能负担）。
- 急救联系卡（详细病史、用药）。

**特殊注意**：

- 不要擅自调整慢病药。
- 急救药品放在易取位置。
- 定期检查慢病药是否过期。

### 过敏体质家庭

**重点加强**：

- 抗过敏药（按医生指导）。
- 肾上腺素自动注射器（医生处方）。
- 食物过敏标签、药物过敏标签。

**特殊注意**：

- 全家人都要知道如何注射肾上腺素。
- 外出携带急救药品。
- 食物来源注意过敏原。

### 宠物家庭

**重点加强**：

- 肥皂水（大量冲洗咬伤用）。
- 消毒剂。
- 创可贴（大面积）。
- 联系狂犬疫苗接种点。

## 八、急救箱的购买建议

### 购买渠道

- **药店 / 药房**：基础药品、器材。
- **医疗器械店**：血压计、AED 等。
- **电商平台**：方便比价，但要选正规商家。
- **红十字会指定店**：质量有保障。

### 品牌选择

- **血压计**：欧姆龙、鱼跃。
- **体温计**：欧姆龙、可孚。
- **AED**：迈瑞、维伟思、ZOLL 等。
- **急救包整体**：3M、海氏海诺等。

### 注意事项

- 药品看有效期。
- 医疗器械看注册证号。
- 不要购买"三无"产品。
- 不要囤积处方药。

## 九、家庭急救箱的使用培训

### 培训内容

- 急救箱位置（全家人都知道）。
- 物品分类（药品 / 器材 / 耗材）。
- 常用物品使用（体温计、血压计、创可贴、绷带）。
- 紧急情况下的取用顺序。

### 培训对象

- 全家人。
- 重点培训 1-2 个家庭成员。
- 老人学习基础使用（体温计、血压计、紧急呼叫）。
- 儿童学习 120 拨打。

### 培训方式

- 家庭演练（详见第 40 章）。
- 厂家使用说明。
- 视频教程。

## 本附录核心要点

1. 基础版必配：体温计、创可贴、消毒剂、退烧药、抗过敏药。
2. 标准版推荐：增加血压计、三角巾、止血带、应急手电。
3. 完整版按需：增加慢病药、AED、血糖仪。
4. 存放：常温、干燥、避光、儿童拿不到、固定位置。
5. 检查：每月一次外观，每季度一次详细，每年一次全面。
6. 急救箱是物化的急救能力，不要等到用时才发现没有。
