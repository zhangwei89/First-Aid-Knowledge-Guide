# 附录七 急救速查手册（可撕页设计）

> 本附录是按可撕页设计的速查手册，每页一个核心场景，可单独撕下放在钱包、急救箱、冰箱上。紧急情况下可立即翻看。

---

## 速查页 1：CPR 速查（成人）

```
┌────────────────────────────────────┐
│  CPR 速查（成人）                  │
├────────────────────────────────────┤
│  适用：意识丧失 + 无呼吸          │
│                                    │
│  1. 安全 → 2. 意识 → 3. 呼救      │
│  4. 呼吸 ≤10 秒 → 5. 按压 30 次  │
│                                    │
│  ★ 频率 100-120 次/分钟           │
│  ★ 深度 5-6 cm                    │
│  ★ 完全回弹                       │
│                                    │
│  6. 开放气道 → 7. 吹气 2 次       │
│  8. 30:2 循环                      │
│  9. AED 立即用                     │
│  10. 持续到专业救援接手           │
│                                    │
│  何时停止：                        │
│  - 专业救援接手                    │
│  - AED 提示不需电击+有呼吸       │
│  - 现场危险                        │
│  - 体力不支                        │
│                                    │
│  民法典 184 条保护合理救助        │
└────────────────────────────────────┘
```

## 速查页 2：AED 速查

```
┌────────────────────────────────────┐
│  AED 速查                         │
├────────────────────────────────────┤
│  1. 开电源键                       │
│  2. 撕电极片                       │
│  3. 贴电极片                       │
│     - 右上：右锁骨下、胸骨右侧   │
│     - 左下：左腋下、肋骨下方      │
│  4. AED 分析（不要触碰病人）      │
│  5. 建议电击：                     │
│     - 确认无人接触                 │
│     - 按下电击键                   │
│     - 立即恢复 CPR 2 分钟         │
│  6. 不建议电击：                   │
│     - 立即恢复 CPR 2 分钟         │
│  7. AED 再次分析                   │
│                                    │
│  儿童可用（用儿童电极片或模式）   │
│  AED 不会让你"做错"               │
│  成人使用无禁忌                   │
└────────────────────────────────────┘
```

## 速查页 3：海姆立克速查（成人 / 儿童）

```
┌────────────────────────────────────┐
│  海姆立克速查（成人/儿童）        │
├────────────────────────────────────┤
│  适用：完全气道梗阻                │
│  - 不能说话                        │
│  - 不能咳嗽                        │
│  - 双手掐脖                        │
│                                    │
│  步骤：                            │
│  1. 站背后，双臂环抱腰部          │
│  2. 一手握拳，拳眼对准            │
│     肚脐上方两横指处              │
│  3. 另一手包住拳头                 │
│  4. 快速向内向上冲击               │
│  5. 重复直到异物排出               │
│     或意识丧失                     │
│                                    │
│  意识丧失：立即 CPR               │
│  婴儿：用拍背 5 次 + 胸部冲击 5 次│
│  自救：用椅子/门框冲击上腹部      │
└────────────────────────────────────┘
```

## 速查页 4：婴儿海姆立克速查

```
┌────────────────────────────────────┐
│  婴儿气道梗阻（1 岁以下）        │
├────────────────────────────────────┤
│  完全梗阻：                        │
│  - 不能哭                          │
│  - 不能呼吸                        │
│  - 脸色发紫                        │
│                                    │
│  步骤：                            │
│  1. 俯卧在前臂上，头低位          │
│  2. 掌根在两肩胛骨之间拍背 5 次   │
│  3. 翻转仰卧在另一前臂            │
│  4. 两根手指在胸骨下半部按压 5 次  │
│  5. 重复直到异物排出或意识丧失    │
│                                    │
│  ⚠️ 不要用腹部冲击                │
│  ⚠️ 意识丧失立即婴儿 CPR          │
│  ⚠️ 不要灌水、灌醋、抠喉咙        │
└────────────────────────────────────┘
```

## 速查页 5：直接加压止血速查

```
┌────────────────────────────────────┐
│  直接加压止血                     │
├────────────────────────────────────┤
│  1. 戴一次性手套（如有）          │
│  2. 干净纱布/布覆盖伤口           │
│  3. 用整个手掌用力按压            │
│  4. 持续 5-10 分钟不松开          │
│  5. 抬高出血肢体高于心脏水平     │
│  6. 出血停止后加压包扎            │
│                                    │
│  ⚠️ 不松、不间断、不换手         │
│  ⚠️ 不撒面粉、香灰、烟丝等土方    │
│  ⚠️ 血液渗透绷带不重包，在外加   │
│                                    │
│  止血带（仅极端）：                │
│  - 宽 ≥ 5 cm                       │
│  - 伤口近心端                      │
│  - 标记时间                        │
│  - 不擅自松开                      │
│                                    │
│  立即 120：                        │
│  - 直接加压 10 分钟不能止血       │
│  - 动脉出血（喷射性）             │
│  - 失血 > 500 ml 出现休克         │
└────────────────────────────────────┘
```

## 速查页 6：烧伤五字诀速查

```
┌────────────────────────────────────┐
│  烧伤五字诀                       │
├────────────────────────────────────┤
│  冲：流动凉水冲洗 15-20 分钟      │
│      （10-15℃，不冰敷）           │
│                                    │
│  脱：在凉水中小心剪开衣物        │
│      （不撕扯，粘住保留原状）     │
│                                    │
│  泡：继续冷水浸泡 10-30 分钟      │
│                                    │
│  盖：清洁纱布/毛巾覆盖            │
│                                    │
│  送：严重者立即送医               │
│                                    │
│  ⚠️ 不涂牙膏、酱油、油、醋       │
│  ⚠️ 不弄破水疱                    │
│  ⚠️ 不冰敷、不火烤                │
│                                    │
│  立即 120：                        │
│  - 面积 > 10%（成人）/ 5%（儿童） │
│  - Ⅲ 度烧伤                       │
│  - 面部、手足、关节、生殖器      │
│  - 吸入性、化学、电烧伤          │
└────────────────────────────────────┘
```

│  - 自行送医（步行、骑车）         │
│  - 等"自己好"                     │
│  - 不记录发病时间                 │
│                                    │
│  时间就是大脑                      │
└────────────────────────────────────┘
```

## 速查页 8：心脏病发作速查

```
┌────────────────────────────────────┐
│  心脏病发作速查                   │
├────────────────────────────────────┤
│  典型：                            │
│  - 胸骨后压榨性疼痛 > 5 分钟     │
│  - 出冷汗                          │
│  - 放射左肩/左臂/下颌            │
│                                    │
│  非典型（老人、糖尿病、女性）：    │
│  - 突发呼吸困难                   │
│  - 上腹不适                        │
│  - 乏力、意识模糊                 │
│                                    │
│  处理：                            │
│  1. 立即 120                       │
│  2. 休息坐位/半卧位               │
│  3. 解开衣领、皮带                │
│  4. 询问备用药：                   │
│     - 硝酸甘油 0.5 mg 舌下含服    │
│     - 阿司匹林 300 mg 咀嚼（无禁忌）│
│  5. 持续观察                       │
│  6. 准备 CPR（意识丧失时）         │
│                                    │
│  血压低者不要含硝酸甘油           │
│  不要步行/骑车去医院              │
└────────────────────────────────────┘
```

## 速查页 9：中暑速查

```
┌────────────────────────────────────┐
│  中暑速查                         │
├────────────────────────────────────┤
│  识别：                            │
│  - 热痉挛：肌肉痉挛、出汗        │
│  - 热衰竭：头晕、面色苍白        │
│  - 热射病：> 40℃、意识障碍、    │
│    皮肤干热（死亡率高）           │
│                                    │
│  处理：                            │
│  1. 阴凉通风处                     │
│  2. 脱衣                            │
│  3. 冷水擦浴 + 风扇               │
│  4. 冰袋大血管：颈/腋/腹股沟    │
│  5. 意识清醒：淡盐水               │
│  6. 意识障碍：不要口服，立即 120  │
│                                    │
│  立即 120：                        │
│  - 意识障碍                        │
│  - 体温 > 40℃                    │
│  - 抽搐                            │
│  - 老人/婴幼儿/慢病               │
│                                    │
│  ⚠️ 不服退烧药、不饮酒、不冰敷    │
└────────────────────────────────────┘
```

## 速查页 10：低血糖速查

```
┌────────────────────────────────────┐
│  低血糖速查                       │
├────────────────────────────────────┤
│  识别：                            │
│  - 出汗、心悸、颤抖               │
│  - 饥饿感                          │
│  - 注意力不集中                    │
│  - 意识模糊、抽搐、昏迷           │
│                                    │
│  处理（意识清醒）：                │
│  1. 立即口服 15-20 克糖分         │
│     - 葡萄糖片 3-4 片             │
│     - 蜂蜜 1 汤匙                 │
│     - 糖水、白糖水                 │
│     - 含糖饮料 150-200 ml         │
│  2. 15 分钟后不缓解再补一次       │
│  3. 缓解后吃正餐/点心             │
│                                    │
│  意识障碍：                        │
│  ⚠️ 不要口服（误吸风险）         │
│  - 复苏体位                        │
│  - 立即 120                        │
│                                    │
│  不确定时按低血糖处理              │
└────────────────────────────────────┘
```

## 速查页 11：过敏性休克速查

```
┌────────────────────────────────────┐
│  过敏性休克速查                   │
├────────────────────────────────────┤
│  三联征：                          │
│  - 皮肤：全身荨麻疹               │
│  - 呼吸：喉头发紧、呼吸困难       │
│  - 循环：心悸、头晕、面色苍白     │
│                                    │
│  处理：                            │
│  1. 立即 120                       │
│  2. 肾上腺素自动注射器（如有）    │
│     - 大腿外侧中部                │
│     - 听到"咔哒"声                │
│     - 保持 3 秒                    │
│  3. 平卧、抬高下肢                 │
│  4. 解开衣领                       │
│  5. 准备 CPR（如意识丧失）         │
│                                    │
│  ⚠️ 双相反应：好转后 4-72 小时   │
│     可能再次发作                   │
│  ⚠️ 所有病人必须送医观察 6-24 小时│
└────────────────────────────────────┘
```

## 速查页 12：触电速查

```
┌────────────────────────────────────┐
│  触电速查                         │
├────────────────────────────────────┤
│  第一步：切断电源                  │
│  - 关电闸、拔插头                 │
│  - 用绝缘物（干木棍、塑料）分开   │
│  ⚠️ 不要直接接触伤者             │
│                                    │
│  第二步：判断                      │
│  - 意识、呼吸                     │
│                                    │
│  第三步：处理                      │
│  - 无呼吸：立即 CPR               │
│  - 意识丧失有呼吸：复苏体位       │
│  - 电烧伤：凉水冲洗 15-20 分钟    │
│  - 立即 120（所有触电者）         │
│                                    │
│  高压电：                          │
│  ⚠️ 不要靠近（专业断电）          │
│                                    │
│  雷电：                            │
│  - 伤者不带电，立即施救           │
│  - 立即 CPR（如需）                │
└────────────────────────────────────┘
```

## 速查页 13：溺水速查

```
┌────────────────────────────────────┐
│  溺水速查                         │
├────────────────────────────────────┤
│  救援原则：岸上优先、不下水优先    │
│  - 抛投漂浮物（救生圈、塑料桶）   │
│  - 长杆、绳索伸过去                │
│                                    │
│  必须下水时：                      │
│  - 会游泳 + 水况熟悉 + 有同伴     │
│  - 携带漂浮物                      │
│  - 从背后接近                      │
│  - 让其头露出水面                  │
│                                    │
│  上岸后：                          │
│  - 先 5 次人工呼吸                 │
│  - 再开始 30:2 CPR                │
│  - 不要控水（倒挂、奔跑）         │
│  - 立即 120                        │
│                                    │
│  ⚠️ 所有溺水者必须送医观察 24 小时│
│  ⚠️ 二次溺水可在恢复后几小时发作 │
└────────────────────────────────────┘
```

## 速查页 14：CO 中毒速查

```
┌────────────────────────────────────┐
│  CO（一氧化碳）中毒速查          │
├────────────────────────────────────┤
│  识别：                            │
│  - 樱桃红色口唇                   │
│  - 头晕、乏力、恶心               │
│  - 意识障碍、昏迷                 │
│  - 多人在同一密闭空间同时发病     │
│                                    │
│  处理：                            │
│  1. 立即开窗通风                   │
│  2. 关闭气源                       │
│  3. 移到空气新鲜处                 │
│  4. 解开衣领、皮带                │
│  5. 立即 120（CO 中毒必须送医）   │
│  6. 必要时 CPR                      │
│                                    │
│  ⚠️ 不要打电话先通风              │
│  ⚠️ 不要在室内开火、开灯          │
│  ⚠️ 所有病人需要高压氧治疗        │
│                                    │
│  预防：                            │
│  - 安装 CO 报警器                 │
│  - 燃气热水器不合规安装           │
│  - 不在密闭空间用炭火取暖         │
└────────────────────────────────────┘
```

## 速查页 17：骨折急救速查

```
┌────────────────────────────────────┐
│  骨折急救速查                     │
├────────────────────────────────────┤
│  识别：                            │
│  - 畸形、肿胀、淤血               │
│  - 骨擦音、异常活动               │
│  - 剧痛、不能活动                 │
│                                    │
│  原则：                            │
│  1. 不移动                        │
│  2. 止血（开放骨折）               │
│  3. 临时固定                       │
│                                    │
│  肢体骨折：                        │
│  - 夹板（木板、硬纸板、卷起的杂志）│
│  - 夹板长度超过骨折两端关节       │
│  - 绑扎不压迫骨折处                │
│  - 检查远端循环                    │
