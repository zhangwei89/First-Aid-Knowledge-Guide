# 第 41 章 急救沟通技巧

## 学习目标

学完本章之后，你能够：

1. 与 120 调度员高效沟通。
2. 与伤者沟通提供心理支持。
3. 与伤者家属有效沟通。
4. 处理围观者、媒体问询。
5. 避开急救沟通的常见误区。

## 适用场景

急救不仅是技术动作，也是沟通协调过程。良好的沟通能让救援更顺畅，能安抚伤者情绪，能减少误解。

## 一、与 120 调度员沟通

**核心原则**：**清楚、简洁、准确**。

**标准沟通模板**：

```
你：您好，这里有紧急情况需要救护车。
    地址：xx 市 xx 区 xx 路 xx 号 xx 小区 xx 楼 xx 单元 xx 室。
    地标：xx 商场东门 100 米。
    病人情况：男，约 60 岁，突然倒地，意识不清，没有呼吸。
    已经做了：现场有人正在做胸外按压。
    我的电话：13xxx。
    请尽快派出救护车！
```

**关键要素**：

- 地址必须精确。
- 地标要明确。
- 病人情况描述清楚。
- 已做处置告知。
- 留电话不挂断。

**不要做的事**：

- 不要先挂电话。
- 不要离开现场。
- 不要给错地址。
- 不要隐瞒病情。
- 不要长时间争辩。

## 二、与伤者沟通

**对意识清醒的伤者**：

1. 自我介绍："我是救护员 / 我学过急救。"
2. 说明情况："我现在要帮你 / 我已经拨打了 120。"
3. 安抚情绪："救护车很快就到 / 你现在很安全。"
4. 取得同意（如果可能）。
5. 解释每一步动作："我现在要给你按压胸口 / 我要包扎你的伤口。"
6. 持续陪伴：说话、观察、反馈。

**沟通技巧**：

- 蹲下来与伤者平视。
- 眼神接触、平静语气。
- 用简单语言。
- 重复关键信息。
- 避免医学术语。
- 保持耐心。

**特殊伤者**：

- 儿童：用温柔的语言、用父母的声音。
- 老年人：尊重、耐心、放慢语速。
- 聋哑人：写字或手势。
- 外语人士：用简单英语或找翻译。
- 醉酒者：减少刺激、避免激怒。

## 三、与家属沟通

**核心原则**：**告知、安慰、配合**。

**告知**：

- 病人情况。
- 已经做的处置。
- 即将要做的处置。
- 120 已经在路上。
- 可能的结果。

**安慰**：

- "我们已经做了该做的。"
- "救护车很快到。"
- "医生会尽全力。"
- 不要过度承诺："没事的 / 一定会好"。

**配合**：

- 询问既往病史、过敏史、用药。
- 通知其他家属。
- 准备身份证、医保卡。
- 联系医生。

**注意事项**：

- 避免指责、推卸。
- 避免医疗判断（不是医生）。
- 避免争辩。
- 给家属情绪出口。

## 四、与围观者沟通

**现场指挥**：

- 指定一人拨打 120。
- 指定一人取 AED。
- 指定一人迎接救护车。
- 维持现场秩序。

**驱散围观**：

- "我需要空间抢救。"
- "请大家后退。"
- "请大家帮忙维持秩序。"
- 不要指责围观者（容易冲突）。

**求助围观者**：

- 寻求帮手（按压、止血、联络）。
- 借急救物品。
- 寻求专业帮助（医生、护士）。

## 五、与媒体沟通

**基本原则**：

- 现场不接受正式采访（除非授权）。
- 简短回答事实性问题。
- 不评论、不推测。
- 引导记者联系官方。

**现场口径**：

- "120 已经在路上。"
- "我们正在全力救援。"
- "具体情况需要等专业人员说明。"

## 六、特殊情况下的沟通

**死亡通知**：

- 由专业人员（医生、警方）通知。
- 施救者不主动告知。
- 不擅自下死亡结论。

**拒绝救治**：

- 清醒的成年人有权拒绝救治。
- 解释风险、尊重选择。
- 记录、签字（病人和见证人）。
- 继续观察、必要时再次建议。

**儿童家长的沟通**：

- 找到家长、告知情况。
- 让家长参与决策。
- 安抚儿童和家长情绪。
- 联系儿科医院。

**多方参与**：

- 统一指挥。
- 角色分工清晰。
- 避免多人下矛盾指令。

## 七、跨文化沟通

**语言障碍**：

- 简单词汇、慢速、重复。
- 身体语言。
- 翻译工具（手机翻译 APP）。
- 寻求翻译帮助。

**文化差异**：

- 尊重对方的信仰和习俗。
- 不强加自己的判断。
- 必要时咨询文化顾问。

## 八、急救沟通常见误区

> **❌ 急救沟通常见错误**
> - 120 通话语无伦次 → 调度员无法定位。
> - 不告诉调度员已做处置 → 信息缺失。
> - 给伤者虚假承诺 → "没事"、伤者反而更不信任。
> - 隐瞒病情 → 干扰医生判断。
> - 与家属争辩 → 浪费时间。
> - 过度解释医学术语 → 伤者听不懂。
> - 大声呵斥围观者 → 引发冲突。

## 九、沟通中的心理调适

**施救者自我调适**：

- 保持冷静（深呼吸）。
- 不被伤者情绪感染。
- 不被家属情绪激怒。
- 不被围观者分散注意力。

**伤者心理支持**：

- 让伤者知道有人在帮助。
- 让伤者知道救援在路上。
- 减少伤者的恐惧、无助感。

**家属心理支持**：

- 让家属知道有人在全力救治。
- 让家属有表达情绪的空间。
- 让家属有具体能做的事（如准备证件）。

## 十、沟通脚本参考

**拨打 120**：

```
120 调度员：您好，120 急救中心。
你：您好，我这里有个人突然倒地，需要救护车。
    地址是 [完整地址]。
    旁边地标是 [明显 landmark]。
    病人 [性别 / 年龄 / 情况]。
    已经做了 [已做处置]。
    我的电话 [号码]。
调度员：好的，救护车已派出，请保持电话畅通。
```

**安抚伤者**：

```
你：我叫 [名字]，我是救护员。
    你现在 [情况描述]。
    我已经拨打了 120，救护车很快到。
    我会一直陪着你。
```

**安慰家属**：

```
你：您好，我是 [身份]。
    您 [家人] 现在 [情况]。
    我们已经 [已做处置]。
    120 已经在路上。
    您可以帮我做 [具体事]。
```

## 本章小结

急救沟通核心：与 120 简洁准确、与伤者温和陪伴、与家属告知安慰、与围观者指挥疏散。语言温和、动作明确、角色分工清晰。

## 何时必须 120

本章为沟通内容，但任何练习中发现真问题立即拨打 120。
