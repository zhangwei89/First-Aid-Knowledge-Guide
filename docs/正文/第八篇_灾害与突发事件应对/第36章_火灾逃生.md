# 第 36 章 火灾逃生

## 学习目标

学完本章之后，你能够：

1. 火灾时正确逃生（低姿、捂住口鼻）。
2. 处理火灾烧伤。
3. 被困时正确求救。
4. 配置家庭防火装备。
5. 避开火灾逃生的常见误区。

## 适用场景

火灾是家庭和公共场所最常见的灾害之一。烟雾是火灾的主要致死原因。本章建立"防烟 + 迅速 + 不坐电梯"的整体思路。

## 一、火灾时的逃生

> 【流程卡】火灾逃生五步
> 1. 大声呼喊 + 立即拨打 119
> 2. 低姿弯腰、捂住口鼻
> 3. 沿安全出口标志撤离
> 4. 不要返回取物、不要乘坐电梯
> 5. 撤离到安全集合点 + 清点人数

详细步骤：

**步骤 1：报警与呼救**

- 大声呼喊"着火了！"通知他人。
- 拨打 119（说明地点、火势、楼层、有无被困人员）。
- 按下手动报警按钮（公共场所）。
- 启动喷淋系统（公共场所）。

**步骤 2：低姿弯腰**

- 烟雾向上走、有毒气体更重。
- 烟气温度：上层 600℃、中层 200℃、下层 50℃。
- 湿毛巾捂住口鼻、压低身体（接近地面）。
- 沿墙边爬行或弯腰前行。

**步骤 3：沿安全出口标志撤离**

- 看绿色疏散指示标志。
- 不要走电梯（可能断电、卡住）。
- 关门防止烟雾蔓延。
- 通过楼梯撤离。

**步骤 4：撤离到集合点**

- 到达室外空旷地。
- 远离建筑物（防止爆炸、坠落）。
- 清点人数。
- 等待救援人员。

## 二、不同场景的逃生

**家庭火灾**：

- 家庭火灾发生 2-3 分钟内是最佳逃生时间。
- 关闭燃气阀门、切断电源（如安全）。
- 沿楼梯撤离。
- 不要跳楼（除非完全被困）。

**公共场所（商场、酒店、KTV）**：

- 进入公共场所先看安全出口位置。
- 火警时按疏散指示标志撤离。
- 听工作人员引导。
- 避免拥挤推搡。

**高层建筑**：

- 立即走楼梯（不要坐电梯）。
- 楼梯间有烟折返、关门等待。
- 在阳台、窗户挥舞衣物求救。
- 拨打 119 说明被困位置。

**车辆火灾**：

- 立即停车、关闭引擎。
- 远离车辆。
- 用灭火器灭火（如可控）。
- 火势失控立即撤离 + 119。

## 三、被困时的求救

**求救方式**：

- 大声呼喊（节省体力、间歇呼喊）。
- 敲击物体（地面、墙壁、金属）。
- 手电、镜子反光。
- 挥舞鲜艳衣物（窗口）。
- 拨打 119 或发送求救短信（12110）。

**等待救援**：

- 关紧门窗、用湿布堵住门缝。
- 在窗口、阳台靠近。
- 节约体力、保持冷静。

## 四、烧伤处理

火场中常见烧伤、吸入性损伤。

**火焰烧伤**：

- 立即灭火（不要奔跑）。
- 脱去燃烧衣物（如果能脱下）。
- 凉水冲洗 15-20 分钟（详见第 13 章）。
- 立即 120。

**吸入性损伤**：

- 面部烧伤、鼻毛烧焦、口腔烟灰、声音嘶哑。
- 半卧位（保持气道开放）。
- 立即 120。

## 五、防火装备

**家庭必备**：

- 灭火器（ABC 干粉灭火器适用）。
- 烟雾报警器（每个房间）。
- 防烟面罩（每人一个）。
- 应急手电 + 备用电池。
- 逃生绳（高层住户）。

**公共场所**：

- 灭火器、消火栓。
- 烟雾报警、自动喷淋。
- 应急照明、疏散标志。
- 安全出口保持畅通。

## 六、灭火器使用

**ABC 干粉灭火器**：

1. 拔出保险销。
2. 一手握紧把手、按压。
3. 另一手握住喷管。
4. 对准火焰根部。
5. 左右摆动喷射。
6. 火焰熄灭后继续喷射 1-2 分钟防止复燃。

**适用范围**：

- A 类：固体可燃物（木、纸、布、塑料）。
- B 类：液体可燃物（油、汽油）。
- C 类：气体可燃物（天然气、液化气）。
- ABC 干粉灭火器适用于大多数火灾。

**不适用范围**：

- 厨房油锅起火（用锅盖）。
- 带电设备起火（先断电）。
- 金属燃烧（D 类）。

## 七、何时必须 120

满足下列任意一条，立即拨打 120：

- 任何烧伤。
- 吸入性损伤。
- 一氧化碳中毒。
- 烟雾吸入后呼吸困难、意识障碍。
- 任何让你"心里没底"的火灾伤情。

## 八、火灾逃生常见误区

> **❌ 火灾逃生常见错误**
> - 乘坐电梯 → 可能断电。
> - 跳楼逃生 → 高空坠落。
> - 重新进入火场取物 → 失去逃生机会。
> - 站立行走 → 吸入烟尘。
> - 不关闭房门 → 烟雾蔓延。
> - 大喊大叫 → 浪费体力。
> - 油锅起火用水浇 → 爆燃。

## 九、家庭防火

**家庭防火 4 件**：

- 燃气关阀。
- 电源切断。
- 烟头熄灭。
- 蜡烛远离可燃物。

**重点隐患**：

- 厨房油烟机定期清洗。
- 电线不私拉乱接。
- 充电不过夜。
- 蚊香远离床铺。
- 儿童不玩火。

## 本章小结

火灾逃生核心：低姿弯腰、捂住口鼻、走楼梯、不坐电梯。烟雾是主要致死原因。烟雾报警器是早期预警关键。

## 何时必须拨打 120

满足下列任意一条，立即拨打 120：

- 任何烧伤。

- 烟雾吸入后呼吸困难。
