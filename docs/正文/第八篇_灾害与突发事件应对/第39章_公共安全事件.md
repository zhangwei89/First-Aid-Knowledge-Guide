# 第 39 章 公共安全事件

## 学习目标

学完本章之后，你能够：

1. 在拥挤踩踏事件中保护自己与他人。
2. 处理爆炸事故的次生伤害。
3. 处理危险化学品泄漏。
4. 在公共场所避险。
5. 避开公共安全事件处理的常见误区。

## 适用场景

拥挤踩踏、爆炸事故、危险化学品泄漏是城市公共安全的主要威胁。本章覆盖这三类事件的应急要点。

## 一、拥挤踩踏事件

**为什么会发生**：

- 人员密度过高（> 4-5 人/m²）。
- 突发恐慌（火灾、警报、谣言）。
- 通道狭窄、楼梯设计缺陷。
- 集体行为失序。

**典型伤亡机制**：

- 压迫窒息（胸廓不能扩张）。
- 挤压伤（内脏破裂、骨折）。
- 踩踏骨折。
- 恐慌摔倒。

**个人保护**：

- 进入拥挤场所前评估风险。
- 注意多个出口、逃生路线。
- 跟随人流前进（不要逆向）。
- 双手抱胸、握拳（保护胸腔）。
- 被挤压时双手护住头颈。
- 跌倒时立即蜷缩、双手抱头。

**现场处理**：

- 立即拨打 110 / 119 / 120。
- 呼喊引导人群疏散。
- 抢救被压人员（先救表层、再救深层）。
- 处理窒息、出血、骨折（详见第 9、11 章）。
- 维持秩序、防止再次拥挤。

## 二、爆炸事故

**常见场景**：

- 燃气爆炸（家庭、餐厅）。
- 工业爆炸（化工厂、矿山）。
- 恐怖袭击。
- 烟花爆炸。

**次生伤害**：

- 冲击波（肺损伤、内脏破裂、鼓膜穿孔）。
- 破片伤（弹片、玻璃、碎石）。
- 烧伤（火焰、热辐射）。
- 建筑倒塌（挤压伤）。
- 烟雾吸入。

**个人保护**：

- 立即卧倒（背朝爆炸方向）。
- 用衣物保护头颈。
- 远离窗户、玻璃、不稳定结构。
- 撤离到安全区。
- 不要围观。

**现场处理**：

- 立即 110 / 119 / 120。
- 评估伤情（按 ABCs 顺序）。
- 处理窒息、出血、休克、骨折。
- 烧伤处理（详见第 13 章）。
- 挤压伤处理（详见第 35 章）。
- 不随意移动伤员（怀疑脊柱损伤）。

## 三、危险化学品泄漏

**常见危险品**：

- 工业气体（液氯、液氨、硫化氢）。
- 油品（汽油、柴油、原油）。
- 农药（有机磷、拟除虫菊酯）。
- 化学品（酸、碱）。

**防护原则**：

- 远离上风向（侧上风向相对安全）。
- 不要在下风向停留。
- 不要触摸不明液体、气体。
- 戴口罩、护目镜（如有）。
- 用湿毛巾捂住口鼻。
- 尽快撤离到安全区。

**撤离路线**：

- 沿垂直风向撤离。
- 不要穿越污染区。
- 避开低洼地（气体密度大时聚集）。
- 到高处（气体密度小时聚集）。

**现场处理**：

- 立即 119 / 110 / 120。
- 评估中毒类型（详见第 18 章）。
- 脱去污染衣物。
- 大量清水冲洗皮肤。
- 不催吐（强酸强碱）。
- 保留毒物样本供医生参考。

**化学品中毒分类处理**：

| 类型 | 处理 |
|------|------|
| 误服强酸强碱 | 不催吐、不中和，立即 120 |
| 皮肤接触 | 清水冲洗 30 分钟以上 |
| 吸入 | 移至空气新鲜处，立即 120 |
| 农药 | 不催吐（部分）、立即 120 |

## 四、公共场所避险

**公共场所应急**：

- 进入公共场所先看安全出口。
- 记住最近出口位置。
- 跟随工作人员指引。
- 避免拥挤区域。
- 关注应急广播。

**学校**：

- 听从老师指挥。
- 紧急疏散有序、不推搡。
- 集合点清点人数。

**商场**：

- 看绿色疏散标志。
- 不乘电梯。
- 不在橱窗旁停留。

**地铁**：

- 听工作人员指引。
- 不挤车门。
- 紧急情况下扳动"紧急开门手柄"（按列车员指导）。
- 使用车厢内的灭火器、紧急报警装置。

**酒店**：

- 进房看逃生路线。
- 紧急情况下低姿撤离。
- 不要跳窗（除非完全被困）。
- 用湿毛巾捂住口鼻。

## 五、个人防护装备

**必备物品**：

- N95 口罩（粉尘、化学品）。
- 护目镜。
- 一次性手套。
- 湿毛巾（应急）。
- 手电 + 哨子。

**公共场所配置**：

- 灭火器。
- AED。
- 急救箱。
- 应急照明、疏散标志。

## 六、心理支持

公共安全事件后，人们可能产生：

- PTSD（创伤后应激障碍）。
- 焦虑、抑郁。
- 恐惧特定场景。

**处理**：

- 心理危机干预。
- 家庭社会支持。
- 必要时专业心理咨询。

## 七、何时必须 120

满足下列任意一条，立即拨打 120：

- 任何意识障碍。
- 心脏骤停。
- 大出血、严重外伤。
- 严重挤压伤。
- 化学中毒。
- 严重烧伤。

## 八、公共安全事件常见误区

> **❌ 公共安全事件常见错误**
> - 拥挤中逆向而行 → 加重踩踏。
> - 跌倒后站立 → 多次被踩。
> - 爆炸后围观 → 二次伤害。
> - 化学品泄漏往下风向跑 → 接触更多毒气。
> - 误服强酸强碱催吐 → 二次损伤。
> - 公共场所恐慌 → 谣言传播、秩序失序。

## 九、应急通讯

**应急号码**：

- 110：报警（治安、刑事）。
- 119：火警、救援。
- 120：医疗急救。
- 122：交通事故。
- 12110：报警短信（听障人士）。
- 12322：地震应急。
- 12119：森林火警。

**应急广播**：

- 关注国家应急广播。
- 关注当地气象、防汛、地震部门信息。

## 十、家庭防灾准备

**家庭应急包（补充）**：

- 饮用水（每人 3 天量）。
- 食物（压缩饼干、罐头等）。
- 急救包。
- 口罩、护目镜。
- 手电 + 电池。
- 哨子。
- 重要证件复印件。

**家庭演练**：

- 拥挤踩踏应急演练。
- 火灾应急演练（详见第 36 章）。
- 地震应急演练（详见第 35 章）。

## 本章小结

拥挤踩踏：双手抱胸 + 跌倒蜷缩 + 双手抱头。爆炸：卧倒 + 保护头颈 + 远离玻璃。化学品泄漏：撤离到上风向 + 不触摸不明物质 + 不催吐（强酸强碱）。
